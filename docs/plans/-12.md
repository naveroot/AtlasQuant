## Scope

PostHog Cloud для product analytics; server-side + client-side tracking ключевых событий; in-app feedback form. Минимальный diff, без watchlist CRUD.

## Analytics Choice

| Критерий | PostHog Cloud | Plausible self-hosted |
|----------|---------------|----------------------|
| Funnels / retention | ✅ built-in | ограничено |
| Event properties | ✅ | базово |
| MVP infra cost | 0 (free tier) | VPS + ops |
| Rails integration | JS SDK + Capture API | snippet only |

**Решение:** PostHog Cloud — лучше для KPI activation/retention и воронки signup → activation.

**Activation proxy:** `view_instrument` (первый просмотр инструмента после signup), т.к. watchlist ещё не реализован (#7 spec out of scope).
