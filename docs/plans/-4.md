# Plan #4: MOEX currency futures quotes and chart

Plane issue: `4d6ec914-1594-4603-8499-e530cafd1596`

**Status:** active

## Scope

Интеграция с MOEX ISS API для валютных фьючерсов FORTS: клиент, сервисы списка и исторических свечей, UI со списком и графиком. On-demand fetch + Solid Cache.

## Data Source (Grounding)

| Данные | MOEX ISS endpoint | Примечание |
|--------|-------------------|------------|
| Список FORTS | `GET /iss/engines/futures/markets/forts/securities.json` | filter by `ASSETCODE` |
| Daily candles | `GET /iss/engines/futures/markets/forts/securities/{SECID}/candles.json?interval=24&from=&till=` | free delayed |
| Settlement history | `GET /iss/history/engines/futures/markets/forts/securities/{SECID}.json` | альтернатива; используем candles |

Валютные `ASSETCODE`: `Si` (USD/RUB dated), `Eu` (EUR/RUB), `Cn` (CNY/RUB), `ED`, perpetuals `USDRUBF`/`EURRUBF`/`CNYRUBF` (`*RUBTOM` asset codes).
