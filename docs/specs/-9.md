# Spec: Разрешить противоречие funding (perpetual) vs базис (MOEX-фьючерсы)

**Status:** active
**Plane:** ac8858bc-3403-4a0f-bf26-716bd704804e
**Created:** 2026-06-30

## Problem

Продукт позиционируется как «расчёт funding rate для perpetual-контрактов», но фактические данные и интеграция (#4) — **MOEX FORTS**: срочные валютные фьючерсы с экспирацией и квази-perpetual контракты MOEX (`USDRUBF`), которые **не используют crypto-style funding rate**. Без зафиксированного ядра ценности нельзя реализовать `FundingCalculator`, метрики базиса и дашборд. Нужно сравнить два варианта ядра, выбрать один и обновить позиционирование в репозитории.

## Acceptance Criteria
