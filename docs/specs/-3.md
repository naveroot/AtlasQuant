# Spec: Заполнить README.md

**Status:** active
**Plane:** 9517f05d-354e-4414-aece-35a4e46c1f93
**Created:** 2026-06-29

## Problem

Корневой `README.md` содержит только Rails-заглушку и не описывает AtlasQuant, стек, установку и команды разработки. Новым участникам и CI-агентам нужен единый onboarding-документ в репозитории.

## Acceptance Criteria

- [x] AC-1: `README.md` содержит раздел «О проекте» с назначением AtlasQuant (отслеживание валютных фьючерсов, расчёт funding rate для perpetual-контрактов) — проверка: `grep -q "AtlasQuant" README.md && grep -q "funding" README.md`
- [x] AC-2: `README.md` перечисляет технологический стек (Ruby 3.2.11, Rails 8.1, PostgreSQL, Tailwind, Hotwire, Solid Cache/Queue/Cable) — проверка: `grep -q "3.2.11" README.md && grep -q "Solid" README.md`
- [x] AC-3: `README.md` содержит пошаговую инструкцию локальной установки (`mise install`, `bin/setup`, `bin/dev`) — проверка: `grep -q "bin/setup" README.md && grep -q "bin/dev" README.md`
- [x] AC-4: `README.md` документирует команды тестирования и CI (`bin/rails test`, `bin/ci`) — проверка: `grep -q "bin/ci" README.md && grep -q "bin/rails test" README.md`
- [x] AC-5: `README.md` содержит раздел о деплое (Docker + Kamal) и ссылку на `AGENTS.md` для MVP scope — проверка: `grep -q "Kamal" README.md && grep -q "AGENTS.md" README.md`
- [x] AC-6: `README.md` не содержит Rails-заглушку «This README would normally document» — проверка: `! grep -q "This README would normally document" README.md`

## Out of Scope

- Перевод `AGENTS.md` целиком в README
- Документация нереализованных MVP-фич (Instrument CRUD, FundingCalculator) как готового функционала
- Создание отдельного `ReadMe.md` (используется стандартный `README.md`)
- Миграция тестов на RSpec (только упоминание целевого стека при необходимости)

## Constraints

- MVP boundaries из AGENTS.md — не описывать OAuth, Redis, real-time exchange APIs как часть продукта
- Содержимое должно соответствовать фактическому состоянию кодовой базы (auth MVP реализован; instruments/funding — planned)
- Язык: русский (как AGENTS.md и остальная документация проекта)
- Минимальный diff — только `README.md` и SDD-артефакты задачи
