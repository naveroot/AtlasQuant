# Spec: Заполнить README.md

**Status:** active
**Plane:** 9517f05d-354e-4414-aece-35a4e46c1f93

## Problem

Корневой `README.md` содержит только Rails-заглушку и не описывает AtlasQuant, стек, установку и команды разработки. Новым разработчикам и агентам нужен самодостаточный onboarding-документ в репозитории.

## Acceptance Criteria

- [x] AC-1: `README.md` содержит краткое описание AtlasQuant (назначение, предметная область: валютные фьючерсы, perpetual, funding rate) — проверка: файл содержит заголовок `# AtlasQuant` и абзац с описанием назначения.
- [x] AC-2: `README.md` перечисляет технологический стек (Ruby 3.2.11, Rails 8.1, PostgreSQL, Solid Cache/Queue/Cable, Tailwind, Hotwire) — проверка: раздел «Технологический стек» с таблицей или списком из ≥6 компонентов.
- [x] AC-3: `README.md` описывает установку и запуск через Mise (`mise install`, `mise exec -- bin/setup`, `mise exec -- bin/dev`) — проверка: раздел «Быстрый старт» с этими тремя командами в code block.
- [x] AC-4: `README.md` документирует основные команды разработки (`bin/ci`, `bin/rails db:prepare`, `bin/jobs`, `bin/rails test`) — проверка: раздел «Команды» содержит все четыре команды.
- [x] AC-5: `README.md` содержит ссылку на `AGENTS.md` для агентов и детального MVP-контекста — проверка: markdown-ссылка `[AGENTS.md](AGENTS.md)` присутствует в файле.
- [x] AC-6: Заглушка «Things you may want to cover» удалена — проверка: строка `Things you may want to cover` отсутствует в `README.md`.

## Out of Scope

- Перевод `AGENTS.md` целиком в README (достаточно ссылки).
- Документация деплоя Kamal пошагово (только упоминание наличия `Dockerfile` / `config/deploy.yml`).
- RSpec-миграция и описание будущего тестового стека (достаточно текущей команды `bin/rails test`).
- Изменение кода приложения, CI-конфигурации или других документов.

## Constraints

- MVP boundaries из AGENTS.md — не добавлять инструкции по Redis, OAuth, биржевым API.
- Содержимое README должно соответствовать фактическому состоянию репозитория (версии из `.ruby-version`, `mise.toml`, `Gemfile`).
- Язык README: русский (как AGENTS.md и docs/).
