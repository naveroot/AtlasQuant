# Spec: Актуализировать AGENTS.md под фактическое состояние кода

**Status:** active
**Plane:** e13ee299-50ee-4f74-8660-4912a49393a6
**Created:** 2026-06-30

## Problem

Разделы «Доменные сущности», «Границы MVP» и «Ориентиры по реализации» в `AGENTS.md` не отражают фактический код: auth (#1), MOEX-интеграция с графиками (#4, #7), deploy/CI (#2) уже реализованы, а AR-модель `Instrument`, `BasisCalculator` и `DashboardController` — нет. Документация вводит агентов в заблуждение при выборе scope и реализации.

## Acceptance Criteria

- [x] AC-1: Раздел «Доменные сущности» в `AGENTS.md` разделяет **реализованное** (`User`, MOEX value object) и **запланированное** (`Instrument` AR, `BasisCalculator`); нет формулировки «все сущности ещё не реализованы».
- [x] AC-2: Чек-боксы «Границы MVP» отражают факт: auth ✅, MOEX list/candles + графики ✅, CI/deploy ✅; `Instrument` CRUD, `BasisCalculator`, RSpec-миграция и аналитический дашборд — ⬜ backlog.
- [x] AC-3: Таблица «Ориентиры по реализации» перечисляет существующие контроллеры (`SessionsController`, `RegistrationsController`, `InstrumentsController` read-only, `HomeController`) и сервисы (`Moex::*`, `Sessions::Authenticate`, `Users::Register`); `DashboardController` и `BasisCalculator` помечены как backlog.
- [x] AC-4: В `AGENTS.md` есть раздел backlog со ссылками на завершённые Plane-задачи (#1, #2, #4, #7, #9) и открытые направления (basis, watchlist, dashboard, RSpec).
- [x] AC-5: `docs/index.md` ссылается на `AGENTS.md` как источник актуального статуса реализации MVP.
- [x] AC-6: `bin/ci` завершается с exit code 0.

## Out of Scope

- Реализация `BasisCalculator`, `Instrument`, `DashboardController`.
- Миграция Minitest → RSpec.
- Изменения Rails-кода, тестов или CI-конфигурации (кроме docs-only правок).
- Синхронизация Plane Pages (manifest обновляется отдельным скриптом).

## Constraints

- MVP boundaries из ADR-001 и текущего AGENTS.md.
- Минимальный diff: только `AGENTS.md`, `docs/index.md`, spec/plan для #11.
- Каждое утверждение в AGENTS.md должно быть проверяемо по пути файла в репозитории.
- Язык: русский (как AGENTS.md).
